### /public 放html文件静态图片和js  分别为img/ js/
### /src/assets 放css文件的静态图片
### 安装
  1.`npm install yarn -g`
  2.`yarn install`

### 启动
  1.配置host地址
    `127.0.0.1       m.zentrust.cn`
  2.`sudo PORT=80 npm start`
  
### yarn 操作
 添加 `yarn add xx` 
 移除 `yarn remove xx`
